# 四、正股策略（核心底仓）

## 1. 入场条件

满足：

```text
Close > MA20
MA20 > MA50
突破 20 日新高
Volume > 1.3 × Avg20
SOXX > MA20
```

解释：

| 条件 | 作用 |
|---|---|
| Close > MA20 | 短期趋势向上 |
| MA20 > MA50 | 中期趋势向上 |
| 突破新高 | 主升浪确认 |
| 放量 | 资金进入 |
| SOXX 强势 | 板块共振 |

---

## 2. 加仓条件

```text
回踩 MA20 后反弹
```

即：

```text
主升浪中的健康回调
```

而不是：

```text
无脑抄底
```

---

## 3. 减仓条件

```text
距离 MA20 > 15%
```

说明：

```text
短期过热
```

---

## 4. 清仓条件

```text
跌破 MA50
```

说明：

```text
趋势结构破坏
```

# 六、期权参数

## 标准参数

| 参数 | 设置 |
|---|---|
| DTE | 28 - 45 天 |
| 行权价 | 当前价上方 5% - 10% |
| Delta | 0.25 - 0.40 |
| 单笔风险 | 账户 1% - 2% |
| 总 Long Call 风险 | 账户 8% - 12% |

---

# 七、滚仓逻辑

例子：

```text
AMD 当前 450

买：
480C
45DTE
```

如果 AMD 涨到：

```text
480
```

则：

```text
卖出旧 Call
滚到：
520C
新的 45DTE
```

# 十四、风险控制（非常关键）

## 单笔风险

```text
<= 账户 1% - 2%
```

---

## Long Call 总暴露

```text
<= 账户 8% - 12%
```

---

## 止损

```text
权利金亏损 40% - 50%
```

直接退出。

---

## 时间止损

```text
买入后 7 - 10 天
没有明显向行权价推进
```

减仓或退出。

---
